# Wage Budget Tracker

A simple web-based budget tracker for logging your wages and expenses.

## Features
- Add wage payments (date, amount, source)
- Add expenses (date, amount, category, description)
- Dashboard showing total income, total expenses, and current balance
- View and delete individual entries
- SQLite database – no setup required
