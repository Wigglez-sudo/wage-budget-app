How to use

A quick guide for first-time setup, daily use, and safer backups.

1. Set up your budget

  1. Pick a cycle: weekly, bi-weekly, every 4 weeks, or monthly.
  2. Set a spending limit for the cycle.
  3. Set a savings target if you want to build cash.
  4. Add or trim categories so they match real life.

2. Use it day to day

  1. Add income when you are paid.
  2. Add expenses as they happen or when bills land.
  3. Review transactions and fix categories when needed.
  4. Check Money check and Reports lab before the cycle ends.

3. Stay safe

  1. Use a strong unique password or 4-word passphrase.
  2. Turn on the timeout PIN if the app may stay open.
  3. Prefer encrypted backups.
  4. Delete plain JSON/CSV exports after use.

    📘 How to use

    Use BudgetVault like a pro

    Quick start

    • Create or rename your budget.
    • Choose the budget cycle that matches how you think about money.
    • Add your regular income first, then your fixed bills, then flexible spending.
    • Set category limits if you want tighter control.

    Useful habits

    • Review uncategorised or unreviewed transactions at the end of each cycle.
    • Use split expense when one purchase belongs to two categories.
    • Use reports to compare this cycle with the last one before making changes.
    • Export an encrypted backup before testing imports or clearing data.

    What the terms mean

    • Spending limit: your target cap for total spending in the cycle.
    • Savings target: what you want left over or moved aside.
    • Rollover: unused category budget can carry forward; overspend can reduce the next cycle.
    • PIN lock: protects an already-open screen after inactivity, not the saved encrypted data.
